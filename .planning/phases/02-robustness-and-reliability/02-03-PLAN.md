<interfaces>
<!-- API endpoints from server.py (Plan 02) that the frontend calls -->

POST /api/preview-csv
  Request: FormData with field "file" (CSV file)
  Response: { "columns": ["col1", "col2"], "numeric_columns": ["col2"], "preview": [{...}, {...}] }

POST /api/start-run
  Request: FormData with fields:
    model_file (File), obs_file (File),
    model_date_col (str), model_value_col (str),
    obs_date_col (str), obs_value_col (str),
    station_name (str), hyperparams (JSON string), seed (int)
  Response: { "status": "started" }

GET /api/run-stream
  SSE stream:
    data: {log line}          — normal log output
    event: done               — pipeline completed
    event: error, data: {msg} — pipeline failed (msg has \n escaped as \\n)
    event: cancelled          — user cancelled

POST /api/cancel
  Response: { "status": "cancelling" }

GET /api/result-summary
  Response: { "status": "done", "metrics": {...}, "output_dir": "..." }
    OR { "status": "no_job" }
</interfaces>

    **Methods on x-data (per D-10, D-11, D-12, D-16, D-17):**

    `pickFile(type, event)` — Per research Pattern 4:
    - Get file from `event.target.files[0]`
    - Store in `modelFile` or `obsFile`
    - POST to `/api/preview-csv` with FormData
    - Populate columns, numeric columns, preview
    - Auto-select first column for date, first numeric for value

    `startRun()` — Per D-11:

    - Build FormData with both files + all form fields + JSON.stringify(hyperparams) + seed
    - POST to `/api/start-run`
    - Set `step = 2`
    - Call `connectSSE()`

    `connectSSE()` — Per D-12, D-13, D-14:
    - `new EventSource('/api/run-stream')`
    - `onmessage`: push `e.data` to `logLines`
    - `addEventListener('done')`: close EventSource, fetch `/api/result-summary`, set `resultMetrics`, set `step = 3`, set `isRunning = false`
    - `addEventListener('error')`: close EventSource, set `errorTrace` (unescape `\\n` to real newlines), set `step = 'error'`, set `isRunning = false`
    - `addEventListener('cancelled')`: close EventSource, set `step = 0`, set `isRunning = false`

    `cancel()` — Per D-16:
    - POST to `/api/cancel`
    - Close eventSource if exists

    `canProceed` — computed-style getter:
    - Returns true when modelFile, obsFile, modelDateCol, modelValueCol, obsDateCol, obsValueCol are all truthy, and stationName is non-empty
