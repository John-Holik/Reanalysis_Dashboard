    **Route 1: GET / (per D-06)**
    Return `FileResponse("static/index.html")`. Use `os.path.join(os.path.dirname(__file__), "static", "index.html")` to resolve the path relative to server.py regardless of working directory.

    **Route 2: POST /api/preview-csv (per D-10)**
    Accept `file: UploadFile`. Read bytes with `await file.read()`. Wrap in `pipeline_bridge._BytesShim(raw_bytes)`. Call `pipeline_bridge.get_csv_columns(shim)`, `pipeline_bridge.get_csv_numeric_columns(shim)`, `pipeline_bridge.get_csv_preview(shim)`. Return JSON:


    **Route 3: POST /api/start-run (per D-11, D-21)**
    Accept:
    - `model_file: UploadFile`
    - `obs_file: UploadFile`
    - `model_date_col: str = Form(...)`
    - `model_value_col: str = Form(...)`
    - `obs_date_col: str = Form(...)`
    - `obs_value_col: str = Form(...)`
    - `station_name: str = Form("MyDataset")`
    - `hyperparams: str = Form("{}")`
    - `seed: int = Form(42)`

    Implementation:
    1. Cleanup previous run (per D-21 / REL-01):
       ```python
       import tensorflow as tf
       tf.keras.backend.clear_session()
       gc.collect()
       if _current_job.get("result") and _current_job["result"].output_dir:
           shutil.rmtree(_current_job["result"].output_dir, ignore_errors=True)
       ```
    2. Read file bytes: `model_bytes = await model_file.read()`, `obs_bytes = await obs_file.read()`
    3. Build DataFrames:
       ```python
       model_df = pipeline_bridge.build_model_df_generic(
           pipeline_bridge._BytesShim(model_bytes), model_date_col, model_value_col)
       obs_df = pipeline_bridge.build_obs_df_dedicated(
           pipeline_bridge._BytesShim(obs_bytes), obs_date_col, obs_value_col)
       ```
    4. Merge hyperparams: `hp = {**DEFAULT_HYPERPARAMS, **json.loads(hyperparams)}`
    5. Create stop_event: `stop_event = threading.Event()`
    6. Launch job: `result, q, t = job_runner.launch_job(model_df=model_df, obs_df=obs_df, variable=model_value_col, station_name=station_name, hyperparams=hp, seed=seed, stop_event=stop_event)`
    7. Update `_current_job` dict with all four values.
    8. Return `{"status": "started"}`

    **Route 4: GET /api/run-stream (per D-12, D-13, D-14)**
    Async generator that drains `_current_job["progress_queue"]` using `get_nowait()`.
    On `stdlib_queue.Empty`: `await asyncio.sleep(0)` and continue.
    Check `await request.is_disconnected()` each iteration.
    Map sentinels:
    - `__DONE__` -> `event: done\ndata: complete\n\n`
    - `__ERROR__` -> `event: error\ndata: {escaped_error}\n\n` (newlines in error_message replaced with `\\n`)
    - `__CANCELLED__` -> `event: cancelled\ndata: cancelled\n\n`
    - Normal message -> `data: {safe_msg}\n\n` (newlines replaced with space)

    Return `StreamingResponse(generator(), media_type="text/event-stream", headers={"Cache-Control": "no-cache", "X-Accel-Buffering": "no"})`.

    **Route 5: POST /api/cancel (per D-16)**
    If `_current_job["stop_event"]` exists, call `.set()`. Return `{"status": "cancelling"}`.

    **Route 6: GET /api/result-summary**
    Return the current job's summary_metrics and output_dir if status is DONE. This endpoint
    lets the frontend fetch results after SSE `event: done`:
    ```python
    @app.get("/api/result-summary")
    async def result_summary():
        result = _current_job.get("result")
        if result and result.status == job_runner.JobStatus.DONE:
            return {"status": "done", "metrics": result.summary_metrics, "output_dir": result.output_dir}
        return {"status": str(result.status.value) if result else "no_job"}
    ```

    **Static mount (AFTER all routes per research anti-pattern warning):**

    ```python
    _static_dir = os.path.join(os.path.dirname(__file__), "static")
    os.makedirs(_static_dir, exist_ok=True)

    ```

    **Main block (per D-03):**
    ```python

        uvicorn.run("server:app", host="127.0.0.1", port=8000, reload=False)
    ```
